# save_delete_messages_vk-2018-
Данный скрипт предназначен для сохранения удаленных сообщений ВКонтакте.
Скрипт каждый момент времени сохраняет полученные сообщения и проверяет их наличие у пользователя.
Если какое-то сообщения не найдено -> его удалили -> скрипт достает его из памяти и отправляет пользователю через специальное сообщество. <br>
<b>На текущий момент, 23.02.2020 он не работает в связи с обновленной год назад политикой ВКонтакте.</b>
